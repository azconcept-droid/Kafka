## Getting started
1. Fork or clone this repo
```
git clone https://github.com/azconcept-droid/kafka.git
```
2. cd into this directory
```
cd kafka
```
3. Install docker
```
bash install-docker
```
4. Start kafka
```
sudo docker compose up -d
```
